函数逼近 vs 曲线拟合
📊
函数逼近 vs 曲线拟合
  • 函数逼近：用简单函数近似代替给定函数
  • 逼近：用连续函数"逼近"连续函数
  • 拟合：用连续函数"拟合"离散数据
  • 逼近/拟合 vs 插值：
    • 逼近/拟合不要求函数通过已知点
    • 插值要求函数必须经过已知点
拟合 vs 插值
🔍
拟合 vs 插值
  • 拟合：整体接近
  • 插值：局部限制
  • 关键区别：
    • 拟合追求整体趋势匹配
    • 插值要求精确通过每个数据点
逼近的度量方法
📏
逼近的度量方法
  • 一致逼近：最大的偏差不超过阈值
  • 公式：‖f(x)-y(x)‖ = max|f(x)-y(x)|
  • 平方逼近：平方和不超过阈值
  • 公式：‖f(x)-y(x)‖ = √∫[f(x)-y(x)]²dx
二次函数拟合示例
🧮
二次函数拟合示例
  • 给定数据点：(1,10), (3,5), ..., (10,4)
  • 拟合函数：y(x) = a + bx + cx²
  • 使用最小二乘法求解系数
  • 建立正规方程组：
    • n·a + b·Σx_i + c·Σx_i² = Σy_i
    • a·Σx_i + b·Σx_i² + c·Σx_i³ = Σx_i y_i
    • a·Σx_i² + b·Σx_i³ + c·Σx_i⁴ = Σx_i² y_i
